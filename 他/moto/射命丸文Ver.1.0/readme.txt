MikuMikuDance用PMDモデルデータ　射命丸文　Ver.1.0

■ 内容

・このデータは東方Projectより「射命丸文」のモデルデータです。
・本データは上海アリス幻樂団の二次創作ガイドラインに沿ってご利用願います。
　　上海アリス幻樂団創作物の二次創作・使用関連ページ
　　http://www.geocities.co.jp/Playtown-Yoyo/1736/t-081.html
・商用利用は禁止です。
・改変・再配布可です。

■ 注意点

・髪とスカートにはIKを入れていません。物理演算の使用を前提としています。
・本モデルは靴底が地面に接地する状態で作っています。
　高下駄の分はモーション読み込み後に全ての親ボーンでY方向に+1程度
　ずらしてやると丁度良くなります。
・羽と扇を別PMDとして同梱しました。ご利用の際には本体と同じモーションを
　読み込んでください。
・カメラ、ノート、ペンを表情モーフで組み込みました。
　ノートは同梱のノート.jpgを書き換える事で文字を記入出来ます。
